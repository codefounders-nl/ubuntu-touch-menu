How to add the ubports menu to your page

Wordpress plugin

Download and install the wordpress plugin:

          Download wordpress plugin

Then add the following tag to your wordpress document: 

          [odoomenu]

Demo: Example of a page that uses the wordpress plugin


Other frameworks

Add the following script tag to your html file, f.e. inside your <head> section:

          <script src="https://ubportsmenu.codefounders.nl/ubportsmenu_min.js"></script>

Demo: Demo Ubports menu as javascript file


Vue component

To use as Vue component add the following inside your <head> section:

          <script src="https://ubportsmenu.codefounders.nl/ubportsmenu-component.js"></script>

and use these tags:

          <ubports-menu></ubports-menu>
Demo: Demo Ubports menu as Vue component
